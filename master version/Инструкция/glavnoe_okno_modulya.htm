1.1. Главное окно модуля

Окно представлено в виде выбора операции какую желаете выполнить

:

Кнопка Заселить человека

Открывает диалоговое окно заселения сотрудника на участок
 
Наверх

Кнопка Высилить человека

Открывает диалоговое выселения сотрудника с участка
Наверх

Кнопка Отобразить список заселенных

Отображает общий список сотрудников с указанием в каком балке (комнате) кто проживает. Так же предоставляет возможность подготавливать документацию для отпуска, списки на ЗП
Наверх

Кнопка Переселение человека

Открывает диалоговое окно переселения сотрудника из одного помещения в другое
Наверх
